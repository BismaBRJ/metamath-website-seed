The two best ways to contact the metamath project are:

  1. Via the metamath mailing list.
  2. By creating a github issue (most often in the set.mm repository but we also list related repositories at other.html and some are in the the metmath github organization).

By asking questions in public you increase the chances of reaching someone with the time and ability to respond.

However, the individuals listed in the CONTRIBUTING file are also willing to be contacted individually.
